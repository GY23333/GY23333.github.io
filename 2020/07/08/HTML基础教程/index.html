HTML基础知识总结。

HTML标签

基本标签

标签 描述 注释
<html> HTML
<body> 主体
<h1>～<h6> 标题 从一级标题到六级标题，由大到小
<p> 段落 浏览器会自动地在段落的前后添加空行，如果在不产生一个新段落的情况下进行换行用<br />
<br /> 换行
<a> 链接 例<a href="http://www.w3school.com.cn">This is a link</a>
<img> 图像 例<img src="XXXX.jpg" width="104" height="142" />
<hr /> 水平线
<q> 短引用
<blockquote> 长引用
<abbr> 缩略词

表格

标签 描述
<table> 表格
<caption> 标题
<th> 表头
<tr> 行
<td> 单元

border="1"则显示边框，border="0"则不显示边框。

列表

标签 描述
<ol> 有序列表
<ul> 无序列表
<li> 列表项
